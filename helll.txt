Project: BlogSphere – Full Stack Blog Management Platform
A multi-user blogging platform where users can create, manage, and interact with blogs, while admins moderate the entire system.
Think of it as a mini version of Medium + Dev.to.

4. Database Design
User Model

{
    username:String,

    email:String,

    password:String,

    bio:String,

    profilePic:String,

    role:{
        type:String,
        enum:["user","admin"]
    },

    createdAt:Date
}

Blog Model

{
    title:String,

    slug:String,

    description:String,

    content:String,

    coverImage:String,

    category:String,

    tags:[String],

    status:String,

    author:{
        type:ObjectId,
        ref:"User"
    },

    likes:[
        {
            type:ObjectId,
            ref:"User"
        }
    ],

    views:Number,

    readingTime:Number,

    createdAt:Date
}

Comment Model

{
    user:{
        type:ObjectId,
        ref:"User"
    },

    blog:{
        type:ObjectId,
        ref:"Blog"
    },

    text:String,

    createdAt:Date
}

Bookmark Model

{
    user:ObjectId,

    blog:ObjectId
}
